# Typescript/Bable/Node Base
Base project for TS / Node.  
TS -> Babel -> JS

Built with,
- TypeScript v.3.6
- Babel v7
- Node v12
- ESLint v6

## Setup
To install dependancies
```
$ npm i
```

## Nodemon
To start nodemon
```
$ npm start
```
## Build
.ts files will be transpiled into the ./dist folder
```
$ npm run build
```

## Run
To run transpiled code
```
npm run prod
```
